# 即梦 Seedance 2.0 分镜提示词生成器

这个 Skill 帮助用户将任何想法转换成 Seedance 2.0 可理解的专业分镜提示词。

## 工作流程

引导用户完成以下步骤：

### 步骤 1：理解用户想法

询问用户：
1. **你想做什么样的视频？**（广告、剧情、MV、特效展示等）
2. **视频时长？**（4-15秒可选）
3. **有什么参考素材吗？**（图片、视频、音频）
4. **想要的风格？**（写实、动漫、水墨、赛博朋克等）

### 步骤 2：分析需求并规划

根据用户的想法，确定：
- **视频类型**：广告片/剧情片/MV/特效展示/其他
- **核心要素**：角色/场景/动作/运镜/特效/音乐
- **素材需求**：是否需要上传参考图/视频/音频
- **技术限制**：是否涉及写实人脸（不支持）

### 步骤 3：分镜设计

将视频按时间轴分解：

```
0-3秒：[开场画面，建立场景]
3-6秒：[发展，主体动作]
6-9秒：[高潮或转折]
9-12秒：[延续或变化]
12-15秒：[结尾，收束画面]
```

每个时间段包含：
- 景别（远景/全景/中景/近景/特写）
- 镜头运动（固定/推/拉/摇/移/跟/环绕）
- 主体动作
- 场景环境
- 光影氛围

### 步骤 4：生成专业提示词

根据分镜设计，生成 Seedance 2.0 格式的提示词：

**基础格式**：
```
[镜头描述]，[主体动作]，[场景环境]，[光影氛围]，[特殊要求]
```

**含素材引用的格式**：
```
@图片1 的角色[动作]，参考@视频1 的[运镜/动作/特效]，[场景描述]，[氛围要求]
```

**时间轴格式**（复杂视频）：
```
0-3秒：[第一段内容]
3-6秒：[第二段内容]
...
```

### 步骤 5：优化建议

提供优化建议：
- 是否需要补充素材
- 提示词可以改进的地方
- 替代方案建议

## 提示词构建技巧

### 1. 素材引用语法

使用 `@` 符号引用上传的素材：
- `@图片1` / `@图片2` ... `@图片9`
- `@视频1` / `@视频2` / `@视频3`
- `@音频1` / `@音频2` / `@音频3`

### 2. 常用专业术语

**景别**：
- 远景、全景、中景、近景、特写

**运镜**：
- 推镜头、拉镜头、摇镜头（左/右/上/下）
- 移镜头、跟镜头（前/后/侧）
- 环绕镜头、手持晃动、一镜到底

**特效术语**：
- 希区柯克变焦、无缝转场、渐变转场
- 粒子特效、黑屏转场、画面渐亮

**光影氛围**：
- 侧逆光、体积光、顶光
- 暖色调、冷色调、电影感、胶片颗粒

### 3. 动作描述技巧

- **具体化**：不说"走路"，说"快步走向门口"
- **序列化**："先...然后...最后..."
- **情绪关联**："愤怒地捶打"、"温柔地抚摸"

### 4. 参考指定技巧

- **角色参考**："@图片1 的人物形象"
- **运镜参考**："参考@视频1 的所有运镜效果"
- **动作参考**："模仿@视频1 的打斗动作"
- **特效参考**："参考@视频1 的转场节奏"

## 特殊场景处理

### 视频延长
明确方向：
- "向前延长 X 秒，内容为..."
- "将@视频1 延长 X 秒"

**注意**：生成长度选择"新增部分"的时长

### 视频编辑（角色替换/剧情改写）
明确指令：
- "将@视频1 中的人物换成@图片1"
- "颠覆@视频1 的剧情，改为..."

### 多模态组合
合理分配素材：
- 图片：角色形象、场景设定
- 视频：运镜参考、动作参考
- 音频：音乐节奏、音色参考

**限制**：总文件数 ≤ 12 个

## 重要限制提醒

⚠️ **必须告知用户的限制**：
1. **不支持写实人脸**：上传包含写实真人脸部的素材会被系统拦截
2. **文件数量限制**：图片≤9，视频≤3，音频≤3，总计≤12
3. **视频时长限制**：参考视频 2-15 秒，生成视频 ≤15 秒
4. **文件大小限制**：图片<30MB，视频<50MB，音频<15MB

## 输出格式

最终输出应包含：

1. **需求总结**：用户的想法提炼
2. **分镜脚本**：时间轴分解
3. **专业提示词**：可直接复制到 Seedance 2.0 使用
4. **素材清单**：建议上传的素材及用途
5. **优化建议**：改进方向和注意事项
